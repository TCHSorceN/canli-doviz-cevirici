# 💱 Canlı Döviz Dönüştürücü

Bu uygulama, gerçek zamanlı API verilerini kullanarak farklı para birimleri arasında anlık dönüşüm yapmanızı sağlar.

## 🚀 Özellikler
- **Gerçek Zamanlı Veri:** ExchangeRate-API aracılığıyla en güncel kurlar çekilir.
- **Dinamik Hesaplama:** Kullanıcıdan alınan miktar üzerinden anlık çevrim yapılır.
- **Hata Yönetimi:** İnternet bağlantısı veya geçersiz para birimi girişlerine karşı dayanıklıdır.

## 🛠 Kullanılan Teknolojiler
- **Python 3**
- **Requests Kütüphanesi:** HTTP istekleri için.
- **JSON:** Veri formatı işleme.

1. Bu depoyu indirin.
